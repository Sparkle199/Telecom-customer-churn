# Data

## Procedure

1. Read the available data summaries, plots, metrics, or error traces.
2. Summarise what they indicate about the data or model behaviour
   (distributions, missingness, class balance, feature relationships,
   metric trends).
3. Suggest, but do not perform, follow-up investigation unless asked.

## Rules

- Distinguish observation (what the data/metrics show) from inference
  (what you think it means) — don't blur the two.
- Do not alter source data files; work from copies/derived artefacts.
